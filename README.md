# Personal Portfolio

This repository contains the source code for my personal portfolio website.

The goal of this website is to showcase my personal projects, technical skills, and experience as a developer. It serves as a central place where I present the work I have done in areas such as software development, game development, and web technologies.

## Features

- Clean and responsive design
- Project showcase
- Skills and technologies section
- Contact information
- Downloadable CV

## Technologies Used

- HTML
- CSS
- JavaScript
- Netlify (for deployment)
